# ecommerce
criar um novo carrinho e inserir produtos nesse novo carrinho:
{
  "produtos": [
    {
      "id": 2,
      "quantidade": 5
    }
  ]
}

atualizar um carrinho com um produto novo ou atualizar a quantidade :
{
  "carrinho_id":12,
  "produtos": [
    {
      "id": 2,
      "quantidade": 5
    }
  ]
}
